Ataque Definicion
Denegación del servicio = INTERRUPCIÓN
DoS (Denial of Service) conocido como Jamming o flooding. Consiste en consumir recursos limitados (Memoria, disco...)
DDoS (Distributed Denial of Service): DoS mediante Bots o botnets
DRDoS (Distributed Reflection Denial of Service) Control de otras máquinas con malware o utilizar otros servidores (inocentes). Ataques:
Connection Flood: Crear muchas conexiones a un servidor WEB
Jamming o flooding con IP spoofing El servidor recibe petis con una IP falsa se le manda información, pero no hay respuest (se satura)
Ping de la muerte Modificar ping (paquete más grande)
E-mail bombing Enviar miles de e-mails para saturar
Smurf o Broadcast storm Broadcast tipo ping con IP falsa, respuesta de miles de PCs a víctima. La variante en UDP se llama Fraggle.
Supernuke o Winuke Mandar paquetes inválidos a puertos Netbios UDP
Teardrop "ataque por fragmentación" saturar el tráfico de la red, aprovechando la fragmentación del protocolo IP.
Hoaxes (bulos) Noticias falsas, Colapsan Servidores de correo mediante envio de noticicas falsas.
Intercepción ≡ Escucha ≡ Monitorización: Afecta a la confidencialidad, Ataques pasivos (análisis del tráfico)
Decoy (Ataques con señuelos) → Interfaces falsas similares a CRTL+ALT+DEL, que solicitan contraseñas.
Keyloggers // Leapfrog (captura las pulsaciones del teclado para robar contraseñas)Ranas capturan contraseñas y direcciones
Snooping downloading Similar a sniffer, pero cogiendo documentos, e-mails, etc.
Eavesdropping Sniffers. roba paquetes. escuchar secretamente, por ejemplo, wifis
Suplantación ≡ Impostura ≡ Fabricación ≡ Enmascaramiento Ataques contra la autenticación y confidencialidad
SPOOFING FALSIFICACIÓN, El intruso se hace pasar por otro usuario, equipo o aplicación.
Hijacking (Secuestro): Cualquier técnica ilegal que lleve consigo el adueñamiento o robo de algo. Secuestro de navegadores.
Criptojacking Secuestro para minar bitcoins: Cryptolocker/Coinhive
IP splicing – Hijacking atacante con sniffer suplanta a cliente real. Se adueña de su sesión.
Piggybacking acceso a una red inalámbrica de Internet con el propio ordenador del intruso (sin permiso dl dueño) ”Robar Internet”. “Entrar en un CPD acompañado”
IP Spoofing Se hace pasar por una entidad distinta. Suplanta la IP. Suplantaciones: IP, arp, DNS, web, correo-e, GPS.
Web spoofing o Defacement Se crea una Web falsa
Watering-holes Uso sin autorización de famosos para publicidad
Mail spoofing E-mail con origen falso → Spam o hoax
Arp Spoofinig, ARP cache poisoning, or ARP poison routing Hace q un equip crea la MAC de una ip local es la de otro equipo de la red, por ejemplo, la puerta de enlace Spoofing-looping: Utiliza usuario/pass Robados para acceder de forma continua (looping) para hacer graves daños.
DNS Sinkhole o Blackhole DNS es un servidor de sistema de nombres de dominio que proporciona información falsa, (sirve para evitar ataques)
DNS poisoning ≡ Pharming = DNS Cache Poisoning: nvenenamiento DNS (datos falsos dominio-IP),
DNS spoofing Es una variante del anterior (DNS poisoning - Pharming) en la que los datos del DNS corrupto se introducen en la memoria caché del servidor de DNS o Man-In-the-Middle DNS Hijacking : El secuestro de DNS o la redirección de DNS, anulando la configuración TCP / IP de un PC para apuntar a un servidor DNS falso
Pharming suplantación de DNS. Suplantación de identidad de un servidor DNS redirigiendo a otra máquina.
WateringHole Estrategia de ataque informático. El atacante quiere atacar a un grupo en particular (organización, sector o región).
Whaling :Ataque de ingeniería social, pishing a ejecutivos de alto nivel
Phishing Es una técnica que se basa en intentar engañar al usuario mediante Ingeniería Social
SMiShing (SMS) Mediante SMS intentar engañar al usuario mediante Ingeniería Social
Vishing (VozIP) Mediante Voip intentar engañar al usuario mediante Ingeniería Social
SpearPhising Phishing dirigido de forma que se maximiza la probabilidad.
Modificación ≡ Manipulación: Afecta a la integridad y Disponibilidad
Tampering o Data Diddling: Modificación desautorizada de archivos o SW
0-Day: Ejecución de código malicioso gracias al conocimiento de vulnerabilidades.
Scumware o escoria cambian los enlaces de los sites o ponen adware encima
Rabbits o conejos gusanos que llenan el disco duro o dejan sin memoria
Rootkit conjunto de software que permite un acceso de privilegio continuo a la víctima. Modifica el comportamiento d un SO xa ocultar su actividad.
Shellshock, también conocida como Bashdoor, es una familia de bugs de seguridad en la ampliamente usada Bash de Shell de Unix
Man In The MIdle (MitM) que en español significa “hombre en el medio”, es un tipo de ataque destinado a interceptar, sin autorización, la comunicación entre dos dispositivos (hosts) conectados a una red. Este ataque le permite a un agente malintencionado manipular el tráfico interceptado de diferentes formas
Ransomware: Restringe el acceso archivos, y pide un rescate a cambio de quitar esta restricción:WannaCry(famoso), Reveton, CryptoLocker, CryptoLocker.F y TorrentLocker, CryptoWall, TeslaCrypt, Mamba, Petya, Locky, Hidden Tear, Cerber (ransomware como servicio), Crysis/Darma, Erebus
Phreaker Usar o abuser del telefono ajeno.
Data Breach Intrusion en un sistema con información confidencial y sin autorización del dueño. Data Leak: fuga de información expuesta y publicada.
Virus, gusanos Malware (crean copias),
troyanos malware que se presenta al usuario como un programa aparentemente legítimo e inofensivo, pero que, al ejecutarlo, le brinda a un atacante acceso remoto al equipo infectado
Bombas lógicas (se ejecutan en evento programado)
Web defacement Malware que modifica portal web
Virtual crab ladilla virtual malware que entra por usar sitios Porno
